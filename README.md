# The Scugog Scoop

A mobile-first static website for local municipal politics and civic issues.

## Pages

- index.html
- council.html
- issues.html
- spending.html
- development.html
- opinion.html
- election.html
- about.html

## Shared files

- styles.css — all design and responsive styling
- app.js — shared navigation, footer and mobile menu
- assets/fairgrounds.jpg — Fairgrounds issue image

## Editing

Edit page wording directly in the relevant HTML file.

The shared navigation and footer are generated by `app.js`, so one edit updates every page.

## Important publishing rule

Do not publish claims about Council, spending, grants, contracts, votes or officials without linking to an official source.

Use the labels consistently:

- Council Update / News
- Public Record
- Analysis
- Opinion

## GitHub Pages

Upload all files to the root of the repository, preserving the `assets` folder.
